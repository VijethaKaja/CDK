Array
 *Supports sequential access and random
 *It supports static allocation and dynamic allocation
 *we can access elements by index
 *it is used for storing and manipuating elements and  to implement stack and queues
Linked List
 *Supports sequential access 
 *It supports static allocation and dynamic allocation
 *we can access elements by next pointers
 *it is used for storing and manipuating elements and  to implement stack and queues
Stack
 *Supports sequential access and random
 *It supports static allocation and dynamic allocation
 *we can access elements by index and pointers
 *it is used for infix to postfix conversion, balancing paranthesis, dfs....
Queue
 *Supports sequential access and random
 *It supports static allocation and dynamic allocation
 *we can access elements by index and pointers
 *it is used for cpu scheduling, bfs
